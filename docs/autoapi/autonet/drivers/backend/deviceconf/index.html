class autonet.drivers.backend.deviceconf.DeviceConf¶

Bases: autonet.drivers.backend.base.AutonetDeviceBackend

Reference implementation of a backend device driver. The driver must implement two methods: get_device and get_device_credentials, which must return an AutonetDevice and AutonetDeviceCredentials object respectively, None if the lookup fails. At present get_device_credentials is not used explicitly by Autonet but there maybe be core changes in the future which will use the method.
